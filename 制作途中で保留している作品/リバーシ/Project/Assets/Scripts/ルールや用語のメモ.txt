// オセロ／リバーシのルール
// 8 × 8
// 初期配置は"クロス"
// 
// ①最終的に数が多いほうが勝ち
//		{
//			盤面がすべて埋まる
//			両者がおける場所がない
//		}
// ②自分の色で相手を挟めたらひっくりかえせる
//		{
//			挟んだ石は全てひっくりかえさないといけない
//		}
// ③おける場所がなければパス
//		{
//			おける場所
//			・縦横斜めのいずれかの方向に、相手の石を挟める場所
//			
//			おける場所がないときのみパスが許される
//		}

//| 盤面		| the board		                            | 一般的に最も使われる表現。ボード全体を指す	|
//| 盤面の状態	| board position / the state of the board	| 石や駒の配置状況を含んだ状態を指す			|
//| 現在の盤面  | current board position                    | 今の局面										|
//| 盤面を制する| control the board                         | 盤面の主導権を握る、という意味の比喩的表現	|

//| 角			| corner									| 盤面の四隅。非常に重要で、取ると強い			|
//| 辺			| edge / side								| 盤面の端の列や行。角に次いで重要				|
//| 手数		| move count								| 打った手の数、または合法手の数				|
//| 合法手		| legal move								| ルール上、打てる手							|
//| パス		| pass										| 打てる手がなく、順番をスキップする			|
//| 終盤		| endgame									| ゲームの終わりに近づいた段階					|
//| 中盤		| midgame									| ゲームの中盤（戦略が重要な部分）				|
//| 終局		| final position / game end 				| ゲームが終了した状態							|
//| 石を返す	| flip the discs / flip the pieces			| 相手の石を自分の色に裏返すこと				|
//| 開幕／序盤  | opening									| ゲーム開始直後の戦略的な手順（定石も含む）    |
//| 定石		| opening strategy / book move				| 序盤の理論的に良い手順（囲碁でも同じ用語が使われる） |

//| 石（白・黒）| disc / disk			                    | 公式ルールでは「disc」が主流（"c" で終わる）	|
//| 石を置く	| place a disc / play a disc			    | 手番で石を置く動作							|
//| 石を返す    | flip a disc								| 相手の石を裏返して自分の色に変える			|
//| 黒番 / 白番 | black's turn / white's turn				| どちらの手番かを示す							|
//| 石の数		| disc count / number of discs				| 最終的な勝敗の判定に使う						|
//| 石を挟む    | sandwich / flip by surrounding			| 相手の石を自分の石で挟んで裏返す行為			|

//例文（英語）
//"Black plays a disc at D3."
//　→ 黒が D3 に石を置いた。

//"White flipped three discs."
//　→ 白が3枚の石を返した。

//"The board now has 32 black and 30 white discs."
//　→ 現在の盤面には黒が32個、白が30個の石がある。
